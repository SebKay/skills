# skills

This repository contains reusable agent skills.

## Install With skills.sh

[`skills.sh`](https://skills.sh/) installs skills with the `skills` CLI. The documented install pattern is:

```bash
npx skills add <owner/repo>
```

To install the skills from this repository, run:

```bash
npx skills add SebKay/skills
```

This uses the [`skills` CLI docs](https://skills.sh/docs/cli) flow and installs the skills published from this GitHub repository.

## Before You Install

- Make sure `npx` is available locally.
- Review the skills on [skills.sh](https://skills.sh/) before installing them.
- Check the skill contents in this repository if you want to verify what will be installed.
